# ACE_tests
This is a test suite for verifying features of the ACE (adaptively compressed exchange) module in the exciting code. To run the tests, do the following.
1. Define which exciting binary you want to run.
2. Set the environment with "ulimit -s unlimited" and "export "OMP_STACKSIZE=1G".
3. Run the tests using "make tests" or check for further options in Makefile.
